# STM32 联调说明

## 联调目标

STM32 通过 UART 周期性发送多通道采样数据，Qt 上位机负责接收、解析、绘图和日志显示。上位机也可以通过命令帧向 STM32 下发控制命令，用于控制 LED、PWM 或采样周期。

## 硬件连接

| STM32 | USB-TTL   | 说明                    |
| ----- | --------- | ----------------------- |
| TX    | RX        | STM32 发送数据到 PC     |
| RX    | TX        | PC 下发控制命令到 STM32 |
| GND   | GND       | 共地                    |
| 3.3V  | 3.3V/不接 | 根据模块供电方式决定    |

注意：UART 通信需要 TX/RX 交叉连接，并保证 STM32 与 USB-TTL 模块共地。

## 串口参数

- 波特率：115200
- 数据位：8
- 校验位：None
- 停止位：1
- 流控：None

## 数据上传示例

STM32 每 20 ms 发送一帧数据：

```text
$DATA,15,123456,25.60,60.20,3.31*7A
```

通道含义可以按实际硬件调整，示例定义如下：

| 通道    | 含义         |
| ------- | ------------ |
| `ch1` | ADC 电压     |
| `ch2` | 温度         |
| `ch3` | 电位器百分比 |

## 上位机控制示例

Qt 上位机可以发送命令帧控制下位机状态：

```text
$CMD,LED,1*2B
$CMD,PWM,500*4F
$CMD,PERIOD,20*61
```

| 命令       | 说明                        |
| ---------- | --------------------------- |
| `LED`    | 控制板载或外接 LED 开关     |
| `PWM`    | 设置 PWM 输出占空比或比较值 |
| `PERIOD` | 设置数据上传周期，单位为 ms |
